# Why Helm?
Imagine deploying an application:

deployment.yaml
service.yaml
ingress.yaml
configmap.yaml
secret.yaml
hpa.yaml

For every environment:

Dev
QA
UAT
Prod

You maintain multiple copies.

# Solution
Helm = Package Manager for Kubernetes

Just like:

apt → Ubuntu
yum → RHEL
pip → Python
npm → NodeJS

Helm manages Kubernetes applications.
